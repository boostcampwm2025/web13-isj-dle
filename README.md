## 프로젝트 소개

부스트캠프 캠퍼들을 위한 최적화된 공간 기반 협업 서비스, Moyo를 소개합니다!!

서비스 이름 'Moyo (모여)'는 한국어로 '모여라', '모여주세요'라는 의미를 담고 있으며, 이는 사람들이 함께 모여 소통하고 협업하는 가상 공간을 제공하려는 서비스의 핵심 가치를 반영합니다. Moyo는 캠퍼들이 하나의 공유된 디지털 공간에서 효과적으로 학습하고 협력할 수 있도록 돕습니다.

### 기획 배경

#### 부족한 공간감과 상호작용
온라인 환경에서는 우연한 만남이나 소그룹 대화가 어려워, 부스트캠프와 같은 팀 기반 학습 환경에서 자연스러운 관계 형성을 방해합니다. 대화 타이밍을 잡기 어렵고 비정상적인 침묵이 발생해 불안감을 유발하며, 사적 공간과 업무 공간의 경계가 흐려지고 고정된 자세로 피로가 누적됩니다.

#### 분산된 협업 도구의 문제점
회의, 코드 수정, 화이트보드 등 협업 기능이 여러 도구로 분산되어 있습니다. 일반 직원은 평균 8.8개, 이사급은 10개 이상의 업무 툴을 사용하며, 업무 중 방해 후 재집중까지 약 23분이 소요됩니다. 이로 인해 잦은 앱 전환이 발생하고 주당 약 4시간의 생산성 손실이 생깁니다.

## Getting Started

### 필요한 기술 스택

- node: v20 이상
- pnpm v10 이상

### 클론

```bash
git clone https://github.com/boostcampwm2025/web13-isj-dle.git
cd web13-isj-dle
```

### 의존성 설치

```bash
pnpm install
```

### shared 패키지 빌드

```bash
pnpm -F @shared/types build
```

### 개발 서버 실행

```bash
pnpm dev:client
pnpm dev:server
```
